# Proposed Dashboard Layout

To achieve a modern, user-friendly look while maintaining the intuitive color-coding, we can shift to a 'bento-grid' style layout. This approach uses cards of different sizes and visual weights, creating a cleaner, more organized interface.

## Layout Plan

1.  **Top Header**: Dashboard Title and summary metrics in a clean, high-contrast bar.
2.  **Main Content Grid**:
    - **Large Cards (Featured Modules)**: For high-frequency modules (e.g., "Generación de Certificados").
    - **Standard Cards (Categories)**: Using the requested color-coded borders to visually group related modules.
    - **Subtle Hover Animations**: Subtle lifts or glows when hovering to increase interactivity.

## Color Palette Maintenance

- **Cursos**: Emerald/Green scale.
- **Participantes**: Blue scale.
- **Facilitadores**: Purple/Indigo scale.
- **Reportes/Certificados**: Amber/Orange scale.

## Implementation Steps

1.  Keep the categories as they are but wrap them in more modern container styling (rounded-xl, subtle border, soft drop shadows).
2.  Add a top-level search/quick access section.
3.  Ensure consistent typography (sans-serif, modern weights).
